# DN5.0-JavaFSE-DeepSkilling

This repository contains hands-on exercises and assignments for the Cognizant Digital Nurture 5.0 Java Full Stack Engineer (FSE) Deep Skilling program.

## Repository Structure

The repository is organized by weeks and modules covering the following topics:

*   **Week 1**: 
    *   Module 1: Design Patterns (Singleton, Factory Method)
    *   Module 2: Algorithms & Data Structures (Search, Financial Forecasting)
    *   Module 3: PL/SQL
    *   Module 4: Logging & Test-Driven Development (TDD with JUnit & Mockito)
*   **Week 2**: 
    *   Module 5: Spring Core & Maven (IoC, Dependency Injection)
    *   Module 6: Spring Data JPA
*   **Week 3**: 
    *   Spring REST (Building RESTful APIs)
    *   JWT Authentication
*   **Week 4**: 
    *   Module 7: Microservices (Eureka Server, API Gateway, Service Communication)
    *   Module 8: Code Quality (SonarQube)
*   **Week 5**: 
    *   Module 9: React
    *   Module 10: GIT
    *   Module 11: Angular

## Technologies Used
*   Java
*   JUnit & Mockito
*   Spring Boot / Spring Data JPA
*   Maven
*   Microservices Architecture
*   React & Angular
*   GIT
